# Transmutation Solitaire

A web-and-mobile-friendly implementation of Transmutation solitaire (rules designed by Hempuli).

Note that drag-and-drop is not supported or needed. Moves are done by selecting the card or stack, then clicking its destination.

Play here: (ad-supported) [[https://www.onlinemathlearning.com/eldritch-invasion-solitaire-game.html](https://www.onlinemathlearning.com/solitaire-transmute-game.html)]

Or here: [https://spheniscine.github.io/solitaire-transmute/]<br>
